Certipedia

As a part of my Engineering Software and Services class at Columbia, I partnered with 3 other students to create a SaaS to store online certifications offered by reputed universities, namely Cornell University, Columbia University and New York University.

The website is made using Ruby on Rails, and the UI is designed using Bootstrap. It is implemented using Agile methodologies and tested using Cucumber.

Users can sort and filter by University and Subject Area, and bookmark courses they are interested in.

FC-SOCIAL

I partnered with two of my classmates to create a social media for our college as a part of our final year undergraduate project.

The website uses PostgreSQL, PHP and jQuery.

Users can create an account and then create and comment on posts. Users can also follow their friends and get suggestions on who to follow. The website also includes a task list.

Learn Valorant!

For my UI Design midterm, I developed a website to learn about the game Valorant
